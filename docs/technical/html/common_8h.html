◆ DEFAULT_COMPARISON_CRITERION

#define DEFAULT_COMPARISON_CRITERION   CRITERION_MORE_RECENT

Default comparison criterion.

This is used to decide whether a file in backup source location is superior to its version potentially present in target location.

Initial value is CRITERION_MORE_RECENT which means that the most recent version of the file will be conserved.
